# node-mocha-chai-typescript-seed

A Node.JS seed with Mocha, Chai and Typescript

## Prerequisites

- Node > 6

## Usage

Install vendor 

    $npm i
    
Build typscript files

    $npm run build
    
Run Tests

    $npm test
